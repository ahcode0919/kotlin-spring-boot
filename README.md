# Spring Boot with Kotlin

Kotlin + Spring Boot API example project

## Setup

### Dependencies

- Homebrew - [Homebrew](https://brew.sh)
- Gradle - `brew install gradle`
- Kotlin - `brew install kotlin`

### Running Project

- Build : `gradle build`
- Run: `gradle bootrun`
- Test : `gradle test`
